запустить mfa_install_env.bat
Скачается conda окружение с установленным mfa и всеми зависимостями

с помощью 1. mfa_start_miniforge.bat сделать json файл с таймингами (нужен txt/srt + audio.wav)

с помощью 2. make_srt.bat сдеать srt файл с таймингами

Новый srt файл перетаскивать не на готовую дорожку с субтитрами а на таймлайн, чтобы создался новый трек. Иначе стили собъются
