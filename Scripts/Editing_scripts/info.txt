
1_CSV_Convert_Columns_to_SRT.bat
	Перетащить и отпустить .csv файл на этот батник 1_CSV_Convert_Columns_to_SRT.bat
	В папке с .csv файлом появятся .srt файлы субтитров собранных из каждой колонки с пропуском 1 строчки (заголовка)
	Колонки с пустым заголовком, а так же содержащие audio и images пропускаются.

2_Audio_Normalize_And_Rename.bat

	Перетащить и отпустить папку с аудиофайлами на этот батник 2_Audio_Normalize_And_Rename.bat , доступно 2 сценария:
	
	1. Нормализация громкости без переименования. 
	Нужно закрыть окно выбора csv файла, тогда файлы будут нормализованы с сохранением их изначальных названий.

	2. Нормализация с переименованием.
	В csv дожна быть колонка с заголовком (первая строчка) name либо speakers. 
	Аудио файлов должно быть столько же, сколько строчек в csv без заголовка
	В том же каталоге, где лежит выбранная папка, должен находиться ровно 1 csv файл. Он будет выбран для работы. Во всех других случаях появится окно с выбором csv файла
	Можно так же запустить bat и вписать в него путь к папке с аудио
	
	Рядом с использованной папкой появится папка "normalized", в ней будут переименованные и нормазизованные аудио файлы
	

3_Convert_Ppro_graphics_txt_to_srt.bat

	В Premiere Pro нужно выбрать графические клипы с текстом (не субтитры). 	
	В окне Text выбрать Graphics -> Export -> Export to text file
	Получившийся файл перетащить на этот bat
	Получится srt файл из графических субтитров, его можно загружать на ютуб

WavSrt_to_animated_srt.bat
	Запустить bat файл, Либо перетащить wav или srt/txt файл на этот bat
	Вписать в него код языка, который будем распознавать.
	Выбрать недостающие файлы (аудио либо текст)
	В папке с текстом появится файл animated_subs.srt в котором будет выполнена подсветка слов
